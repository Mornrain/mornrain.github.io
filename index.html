四月

8 日

  • Our lives were a cycle—the cycle of day, the cycle of the seasons—circles of perpetual change that

    —《当你像鸟飞往你的山》

  • 对话的目的是什么？

    ​ 自己还是会受到主观意识的影响，根据自己对这个人的认识而决定自己对其言说内容的可信程度，并时常聊着聊着发觉讨论并无意义而开始变得沉默不语。

    ​ 每个人讲话都有其自身的出发点与角度，不能因为对方讲的都是专业内容就认为正确而听之任之，也不能因为抓住一两处错误就认为荒唐而全盘否定，前后推导得出这些话的因果关系也能帮助自己有新的收获。

    ​ 对话的目的还是帮助我们对世界有更全面的认知，认知世界需要一定的观点，我们从日常生活中的所见所闻收获了零零星星的认知片段，如何将这些系统的形成自己的认知方法论，变成自己对世界的理解，以及后续可以作为自己的价值观清晰、有条理的进行叙述输出，我们思考后得出结论，然而这个结论还需要通过实践得到检验，同时确保自己不会在错误的方法道路上越走越偏。实践的最好方法之一莫不过是与人交谈。与人交谈，我们可以获取不同人对同一事件经过其价值观整理出的看法，他讨论的事情出发点是什么，如何得出这样的结论等等，这都是事后值得我们反思的。

    ​ 对话的最终目的应该是思想上的碰撞，我虽然而且也不必信服你的整个理论，你的整个态度，但我能从这场对话中修正了我的某些观点，从而深化自己，那便是最终意义。

在 Create React App 中使用 Typescript

项目配置

Create React App 内置了对 TypeScript 的支持。

需要创建一个使用 TypeScript 的新项目，在终端运行：

npx create-react-app my-app --template typescript

下载 React Router：

npm install react-router-dom@6

知识点

  • React 项目会以 public 文件夹为根目录
